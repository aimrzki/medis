Register Akun Dokter

URL

/api/doctor/signup

Method

POST

Lakukan verifikasi pada tombol yang dikirim ke email (cek spam)

Request Body - Raw json

  {
        "first_name" : string,
        "last_name" : string,
        "username" : string,
        "contact_number" : string,
        "password" : string,
        "email" : string
  }

Response

    {
        "code": int,
        "id": int,
        "message": string,
        "token": string
    }

Menambahkan Medical Record Pasien

URL

/api/doctor/medical-record

Method

POST

Request Body - Raw json

    {
            "patient_name" : string,
            "birth_date" : string,
            "email" : string,
            "phone_number" : string,
            "diagnosis" : string,
            "prescription" : string,
            "care_suggestion" : string
    }

Note

Include authorization token in headers (Bearer token from doctor auth login)

Response

    {
        "code": int,
        "data": {
            "id": int,
            "patient_name": string,
            "birth_date": string,
            "email": string,
            "phone_number": string,
            "diagnosis": string,
            "prescription": string,
            "care_suggestion": string,
            "doctor_id": int,
            "created_at": string,
            "UpdatedAt": string
        },
        "error": false,
        "message": "Medical record created successfully"
    }
